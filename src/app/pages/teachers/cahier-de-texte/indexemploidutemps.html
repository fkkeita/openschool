Modifiez les matières & professeurs pour n’importe quelle semaine
Semaine du ...
Fonctionnalités : Vous pouvez modifier chaque case (matière, professeur) pour la semaine sélectionnée. Les changements sont sauvegardés dans le même stockage que le cahier de texte principal. Cela permet d’anticiper les indisponibilités d’enseignants pour les semaines à venir.
Les signatures déjà effectuées sur des semaines passées ne sont pas altérées.
Les modifications s’appliquent uniquement à la semaine choisie.
